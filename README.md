## White Panda - Dashboard
-----

Responsive dashboard built completely in [React](https://reactjs.org/ "ReactJs")

### Steps to run the project
 

 1. After unzipping, cd into panda_dashboard and install dependencies.
 
       `cd panda_dashboard`

       `npm install`

 2. After installing dependencies, run the app by following commands.
    
       `npm start or yarn start`
    
 3. Navigate to http://localhost:3000 on the browser.
